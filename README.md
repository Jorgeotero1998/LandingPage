# Landing Page con React

Proyecto desarrollado como parte del Bootcamp de 4Geeks Academy. Esta es una landing page construida íntegramente con **React** y **Bootstrap 5**, siguiendo principios de diseño responsivo y componentes funcionales.

## Características:
* **Componentes Funcionales**: Uso de Hooks y funciones para una estructura moderna.
* **Propiedades (Props)**: Implementación del principio **DRY** (Don't Repeat Yourself) mediante un componente dinámico para las tarjetas.
* **Bootstrap 5**: Estilos exactos basados en el template "Start Bootstrap".

## Tecnologías utilizadas:
* React.js
* Bootstrap
* Vite (Entorno de desarrollo)
